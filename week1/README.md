## 1. 로그인, 로그아웃, 회원가입 구현

    - 로그인 상태는 PHP session을 사용해 관리
    - 로그인, 회원가입 요청은 POST method 사용
    - 유저 정보는 모두 MySQL로 관리

## 2. 게시글 쓰기, 읽기, 수정, 삭제 기능 구현

    - 게시글 - 제목, 내용, 작성자 정보
    - 게시글 작성은 로그인 상태에서만
    - 게시글은 작성자만 수정, 삭제
    - 게시글 쓰기, 수정, 삭제는 POST method 사용
    - 게시글 정보는 모두 MySQL로 관리

우선 기능 구현에 초점을 맞추고, 추후 보안까지 고려한 구현을 추가로 해볼 예정
